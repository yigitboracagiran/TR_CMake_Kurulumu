# CMake Kurulumu

NOT: Bu adımlar Linux tabanlı bir işletim sistemi için geçerlidir.

Not: Daha önceden kuruluysa `sudo apt-get purge cmake` ile kaldırınız.

1- `git clone https://github.com/Kitware/CMake.git`

2- `cd CMake/`

3- `mkdir build`

4- `cd build/`

5- `../bootstrap`

6- `make`

7- `sudo make insall`
